# QR Generator

A Java Maven application for generating QR code images from text or URLs.
The system encodes input data and produces QR codes as PNG image files using
a Java QR code library.

---

## Features

- Generate QR codes from text or URLs
- Save QR codes as PNG image files
- Multiple QR code generation support
- Backend-only Java application
- Simple and lightweight implementation

---

## Tech Stack

### Backend
- Java

### Database
  - MySQL

### QR Code Library
- ZXing (Zebra Crossing)

### Build Tool
- Maven
